# COVID-19 Vaccine Effectiveness Analysis

A Python-based data visualization project analyzing COVID-19 vaccine effectiveness trends, age-group variations, and safety profiles using data from 2021-2025.

## Features
- **Waning Immunity Analysis**: Visualizes how protection changes over 20+ weeks.
- **Demographic Comparison**: Effectiveness data for children, adults, and high-risk groups.
- **Safety Profile**: Correlation between dose number, reaction frequency, and effectiveness.

## Requirements
- Python 3.x
- Matplotlib
- Numpy

## Installation
```bash
pip install matplotlib numpy
python analysis.py
